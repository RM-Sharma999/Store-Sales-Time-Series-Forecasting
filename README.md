# Store Sales Time Series Forecasting
Accurate sales forecasting helps retailers optimize inventory, reduce waste, and make informed business decisions. This project uses the Corporación Favorita Store Sales dataset from Kaggle, which includes historical sales records along with promotions, holidays, transactions, oil prices, and store information.

The goal of this project is to analyze sales patterns and forecast future demand across product families. Using exploratory data analysis, feature engineering, and time series forecasting techniques, the project identifies key sales drivers and generates actionable forecasts.
